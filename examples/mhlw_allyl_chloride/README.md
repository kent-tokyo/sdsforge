# Example: MHLW Sample SDS — Allyl Chloride (塩化アリル)

This example converts the official MHLW sample SDS for allyl chloride into
MHLW SDS data exchange JSON format.

## Source / 出典

厚生労働省「資料5-4別紙 塩化アリルモデルSDS」  
https://www.mhlw.go.jp/content/11201000/001371593.pdf

出典：厚生労働省ホームページ（PDL1.0）  
上記資料をもとに sdsforge の変換テスト・デモ用に加工して作成。  
This fixture was generated by converting the above MHLW sample SDS
into MHLW SDS data exchange JSON for demonstration and testing purposes.

The source PDF is not redistributed in this repository.
Run `convert.py` to download it and reproduce the output.

## Run

```bash
pip install sdsforge         # or: pip install -e ../../sdsforge_py
export ANTHROPIC_API_KEY=sk-ant-...
python convert.py
```

This downloads the PDF from the MHLW website, verifies its SHA-256,
converts it to MHLW JSON, and writes `expected.json` and `expected_report.json`.

## Files

| File | Description |
|------|-------------|
| `source.url` | URL of the source PDF on the MHLW website |
| `source.sha256` | SHA-256 of the source PDF (for integrity check) |
| `convert.py` | Download + convert script |
| `expected.json` | Expected MHLW SDS JSON output |
| `expected_report.json` | Expected ConversionReport output |
